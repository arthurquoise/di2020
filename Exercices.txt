Exercice générique Pile
Créer une classe générique Pile
La classe pile a :
    - Un tableau d'éléments.
    - Une méthode pour empiler un élément.
    - Une méthode pour dépiler.
    - Une méthode pour récupérer l'élément à l'emplacement i.
    - Une méthode rechercher qui accepte comme paramètre la méthode de recherche.

Exercice List
Dans le TP employé, il faut remplacer l'utilisation des tableaux par des listes.


Exercice Event
Ajouter un event Adecouvert dans le compte bancaire, cet event doit indiquer le solde et le numéro de compte


Exercice ADO.NET

Ecrire un programme qui permet d'ajouter, suppirmer, afficher à partir d'une base de données des contacts
(Chaque contact a un id, nomn prenom, téléphone)


Exercice 1 ASP.NET CORE

Créer l'architecture d'une application web qui permet gérer les contacts
(Page pour afficher les contacts, une page pour afficher un contact, une page pour ajouter un contact)

- Sur la page d'accueil, affichez une liste de contacts récupérée à partir d'une base données.
- Pour chaque contact ajoutez un lien pour afficher le détail du contact, ainsi qu'un lien pour supprimer le contact.
- Ajoutez un formulaire pour ajouter un contact


Exercice 2 Compte Bancaire ASP.NET CORE

- Réalisez une application web qui permet de créer un compte bancaire.
- Sur la page d'accueil, on peut effectuer une recherche par numéro de compte, on affichera les informations du compte
après recherche (client, solde, listes opérations).
- Après la recherche d'un compte, on peut effectuer un dépôt ou un retrait.
